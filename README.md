## ğŸš€ Installation & Local Setup

Want to run this portfolio locally? Follow these simple steps:

### Prerequisites

Make sure you have the following installed:
- A modern web browser (Chrome, Firefox, Safari, Edge)
- [Node.js](https://nodejs.org/) (optional, for development server)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Karam-Samer/Karam-Samer-Portfolio.git
   cd Karam-Samer-Portfolio
   ```

2. **Option A: Open directly in browser**
   - Simply open `index.html` in your web browser
   - Double-click the file or right-click â†’ "Open with" â†’ your browser

3. **Option B: Run with a local server (recommended)**
   ```bash
   # Install dependencies (one-time setup)
   npm install

   # Start the development server
   npm run serve
   ```
   - Open your browser and navigate to `http://localhost:8080`

4. **Optional: Run Lighthouse performance tests**
   ```bash
   # Desktop performance test
   npm run lighthouse:desktop

   # Mobile performance test
   npm run lighthouse:mobile
   ```
